# Examination Questions

## Availability

The USAWC oral comprehensive examination uses a slate of 12 questions covering all seven Program Learning Outcomes (PLOs). The question slate rotates annually.

**AY25 questions:** Pending approval for public release. Once the AY25 slate is no longer in active use and release is approved, the questions will be posted here with the PLOs they address.

**AY26 questions:** Currently in use. These will not be published until they rotate out of the active examination cycle.

## For Replicating Institutions

You do not need USAWC questions to run MilBench. Use your own institution's capstone examination questions. The protocol requires only that questions:

1. Demand integration across at least two disciplines in the curriculum.
2. Are open-ended and require sustained analysis (not factual recall).
3. Allow for impromptu faculty follow-up questioning.

If you wish to compare results directly with the USAWC pilot, contact the authors to discuss question alignment.

## Question Format

When published, questions will follow this format:

```
## Question [Number]

[Question text]

**PLOs addressed:** [List of Program Learning Outcomes]
**Disciplines:** [Core courses or subject areas the question draws from]
```
